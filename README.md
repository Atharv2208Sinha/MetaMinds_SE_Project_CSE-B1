# MetaMinds_SE_Project_CSE-B1

## Smart Inventory and Sales Analytics System for Small Businesses

This project provides a simple, efficient, and reliable software solution for managing inventory, sales, and basic business analytics for small-scale enterprises such as kirana stores, retail shops, pharmacies, and wholesalers.

### Features

- *Inventory Management*: Add products with details like name, description, category, price, and quantity.
- *Sales Tracking*: Record sales transactions and update inventory automatically.
- *Basic Analytics*: View sales revenue trends with simple charts.

### Technology Stack

- Backend: Python with Flask
- Database: SQL
- Frontend: HTML, CSS
- Analytics: Matplotlib for charts

### Installation
   
1. Install dependencies:
   
   pip install -r requirements.txt
   
  
2. Run the application:
   
   python app.py
   

3. Open your browser and go to http://localhost:5000

### Usage

- Navigate to the home page to view current inventory.
- Add new products via the "Add Product" page.
- Sell products by clicking "Sell" on a product row.
- View analytics on the "Analytics" page.
